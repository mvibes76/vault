# Video Vault v27 Notes

## Sidebar tree
- Sidebar folders now render as a parent/child tree.
- Root folders stay as main folders.
- Nested galleries/folders show indented under their parent with a collapsible dropdown.
- Active child galleries auto-expand their parent folder.
- Root folders get a quick `+` action to create a gallery inside that folder.

## Web-fluid interaction
- Reworked the white fluid action into a smaller bottom-right squirt.
- The effect now travels from the button toward the media instead of exploding as a huge centered icon.
- Added a smooth glass-like white streak with subtle drips.
- Kept the button isolated from playback controls and easy to remove if needed.

## Build
- `npm install` completed.
- `npm run build` passed.

## Schema
- No new Supabase schema changes in v27.
- v26 schema is still current.
